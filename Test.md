# Markdown Cheatsheet

## Headings
# H1
## H2
### H3
#### H4
##### H5
###### H6

---

## Emphasis
*italic* or _italic_
**bold** or __bold__
***bold and italic***
~~strikethrough~~

---

## Lists

**Unordered list:**
- Item
- Item
  - Nested item
* Another item

**Ordered list:**
1. First item
2. Second item
   1. Nested item

---

## Links
[Link Text](https://example.com)

---

## Images
![Alt Text](https://example.com/image.png)

---

## Code
**Inline code:**
Here is some `inline code`.

**Code block:**
```
function helloWorld() {
  console.log("Hello, world!");
}
```

**Code block with language:**
```js
const greet = () => console.log("Hi!");
```
